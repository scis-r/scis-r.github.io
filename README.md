# scis-r.github.io

Hello and welcome to my personal website!

  

## About This Project
I started this project because I wanted to get into web development. I realized one day that having my own website with links to all of my projects I've been a part of over the years would be fun, but I didn't want to use a "website creator" site like Wix or Squarespace. For one, I'm a programmer, and using tools like those kind of seemed like cheating in my eyes. If I'm going to have a website that displays my coding projects, I might as well make the website itself another one of my projects. Also, using tools like those would cost me more money than it would to just buy a domain name, and in the long run, wouldn't give me the highest level of customization. So, I booted up VS Code and made my first html file, and now we're here.

I've made this a repo on GitHub so I can use GitHub Pages to host the site while I work on it. I figure there isn't much point of buying a domain before the site is close to being finished.

I'll keep this repo updated until I decide I've done enough to buy my own domain. Then, this repo will live on in my profile as I update the site more from the actual domain.
